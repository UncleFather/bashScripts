# Bash
## bashScripts
### _Скрипты на bash для облегчения задач администрирования_
+ __Automate_Astra.sh:__ 
__Настройка нового рабочего места с предустановленной операционной системой Astra Linux (Orel) 2.12.13:__ ([подробное описание](http://manaeff.ru/forum/viewtopic.php?p=2345))
Скрипт в процессе выполнения перезагружает компьютер, а так же осуществляет выход пользователя из системы, поэтому его необходимо запускать несколько раз. С целью упрощения процесса, начальный скрипт необходимо создать на рабочем столе под именем Automate_Astra01.sh. После запуска, он создаст свою копию с именем aa.sh, которую нужно будет запускать после каждой перезагрузки до окончания процесса настройки (всего его нужно запустить 4 раза). В копии скрипта отработанные строки будут удаляться по завершении очередного этапа настройки для того, чтобы при повторном запуске не выполнять уже отработанные команды. Для применения эталонных настроек браузера Google Chrome необходимо положить файл с этими настройками _Preferences_ в одну папку со скриптом. Файл можно скопировать с предварительно настроеннного браузера на другом компьютере.   
__Скрипт выполняет следующие задачи:__
  - Установка пакетов:
    - dos2unix
    - anydesk
    - GoogleChrome
    - skype
    - wine
    - playonlinux
  - Обновление всех ранее установленных пакетов
  - Очистка системы и локального репозитория от лишних файлов
  - Создание нового пользователя admin с паролем MyPass123 и добавление его в нужные группы
  - Установка время блокировки учетной записи (при срабатывании блокировки после неудачных попыток ввода пароля) равным 1200 секунд
  - Копирование на рабочий стол ярлыков:
    - GoogleChrome
    - Skype
    - Libreoffice Writer
    - Libreoffice Calc
    - Termial Fly
  - Настройка браузера GoogleChrome в соответствии с заранее заданными в файле Preferences предпочтениями
  - Установка часового пояса Asia/Yekaterinburg и запуск синхронизации времени
  - Деактивация функции энергосбережения (кроме отключения монитора)
  - Изменение имени компьютера на заданное в командной строке
  - Установка агента администрирования Касперского (параметры будут вводиться с клавиатуры)
